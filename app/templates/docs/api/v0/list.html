Gallery list API

The gallery list API allows for the fetching of lists of extensions. It is a temporary addition until the search API is completed.

Send a request to the following URL:

GET {{ baseURL }}/api/v0/list.json?parameters

Parameter Description
type The type of extension to list (“gadget” or “robot”). Leaving this blank will return all extension types.

The response will contain an Array of Objects, each of which has the same properties as a result from the extension info API.
