Search API

The search API allows for the fetching of search results from the gallery.

Send a request to the following URL:

GET {{ baseURL }}/api/v0/search.json?parameters

Parameter Description
q The search query. This can be any query a user could type into the search box on the Wave Extensions Gallery site, which can include advanced search operators
limit The limit on the number of extensions to return. The default value is 20
offset The number of documents to skip in search results. The default value is 0.
sortby How to order the results. Valid values are currently match (sort by closest match to the query) and rating (sort by highest rating). The default value is match.

The response will contain an Array of Objects, each of which has the same properties as a result from the extension info API.
