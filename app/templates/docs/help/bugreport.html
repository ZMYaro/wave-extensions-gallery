Report an issue

Have you noticed something in the extension gallery misbehaving, not looking right, or displaying wrong information? Report the issue so people know it exists and can fix it. Follow the instructions below to report the issue.

Note: you must have a GitHub account to file issues. If you do not have a GitHub account, you can sign up for free here.

  1. Navigate to github.com/zmyaro/wave-extensions-gallery/issues/new in your browser.
    • You will be prompted to sign into your GitHub account if you have not done so already.

  2. Give a short summary of the problem in the “Title” field, and type more information in the “Leave a comment” section.
    • It is generally helpful to mention which web browser you are using in the “Leave a comment” area.

  3. Click the green “Submit new issue” button.
