Advanced search operators

You can find extensions just by searching for some relevant words, but if you are looking for extensions of a specific type or developed by a specific person, advanced search operators can help you narrow down your results.

Operator Definition Examples
developer: Used to specify the creator of the extension (by the nickname he/she is listed under in the gallery Example: developer:zmyaro
Meaning: Extensions created by zmyaro
type: Used to specify the type of extensions to return Example: type:gadget
Meaning: Only show extensions that are gadgets
category: Used to specify a category from which to return extensions

Choose from the following: Game, Image, Map, Music, Productivity, Search, Team, Time, Travel, Utility, Video, Voice, Voting, Other 		Example: category:voting
Meaning: Only return voting gadgets
