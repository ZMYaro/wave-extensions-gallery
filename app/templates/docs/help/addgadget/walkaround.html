Add a gadget to a wave in Walkaround

  1. If the blip where you want to add the gadget is not already in edit mode, click the “Edit” button (or press Ctrl+E) to put it in edit mode.


  2. Click the add gadget button ().


  3. In the pop-up that appears, click in the text box labeled, “Enter a custom gadget URL”. In that text box, paste {% if gadgetURL %}“{{ gadgetURL }}”{% else %}the gadget URL{% endif %}.
