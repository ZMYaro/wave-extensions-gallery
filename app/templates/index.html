Welcome!

This is an incomplete version of the Wave Extensions Gallery, which will, eventually, provide and index of wave extensions and hosting for wave gadgets.

Below is a list of things you can try out:

  • Navigate to /about to view information about wave and wave extensions
  • Navigate to /dev to view your developer dashboard, from which you can:
    • See a list of extensions you have added
    • Add new extensions
    • Edit extension entries
    • View extension pages in the gallery
  • Navigate to /gallery, /gallery/gadgets, or /gallery/robots to browse the gallery (NOT YET IMPLEMENTED)
  • Navigate to /gallery/[extension ID] to view an extension's info page
  • Navigate to /account to view your account information, including your starred extensions (NOT YET IMPLEMENTED)
